Welcome to Polyphone documentation!
===================================

Polyphone is a free and open-source software for editing sf2, sf3, sfArk and sfz soundfonts.
A clean interface and convenient tools have been implemented to efficiently deal with small or big instruments.
Polyphone is available in different languages for Windows, Mac OS X, Linux and comes with a forum and a detailed documentation.
