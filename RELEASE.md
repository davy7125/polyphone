# Information for users

## Code signing policy

Free code signing provided by [SignPath.io](https://signpath.io/), certificate by [SignPath Foundation](https://signpath.org/).

## Privacy policy

This application uses client-side HTTP requests exclusively for operational purposes, including:

* application version checks and update retrieval,
* downloading the soundfont list and associated soundfont files,
* user authentication and account access.

The application does not collect, track, or share any personal data.
No data is transmitted to any server other than [https://polyphone.io](https://polyphone.io), and no third-party services are involved.
