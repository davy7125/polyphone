La barre de menu contient les éléments suivants :

  • Fichier
    • Nouveau
      Crée un nouveau fichier .sf2 vierge.
    • Ouvrir
      Ouvre un fichier .sf2 existant.
    • Enregistrer
      Enregistre le fichier actuel.
    • Enregistrer sous
      Enregistre le fichier sous un autre nom.
    • Importer des sons
      Importe des sons au format .wav.
    • Exporter des sons
      Exporte des sons présents dans le fichier .sf2 au format wav.
    • Exporter (sfz)
      Exporte le fichier courant dans le format sfz. Les modulateurs ne sont pour l'instant pas pris en charge.
    • Fermer le fichier
      Ferme le fichier actuel.
    • Quitter
      Quitte le logiciel.
  • Edition
    • Annuler
      Annule la dernière action effectuée.
    • Rétablir
      Rétablit la dernière action annulée.
    • Copier
      Copie un élément, que ce soit dans l'arborescence ou dans un tableau.
    • Coller
      Colle un élément copié.
    • Supprimer
      Supprimer un élément sélectionné, dans l'arborescence ou dans un tableau.
    • Renommer / Renommer en masse
      Renommer un élément sélectionné dans l'arborescence. Possibilité de renommer plusieurs sons simultanément.
    • Préférences
      Ouvre les préférences du logiciel.
  • Outils
    • Affiche les différents outils disponibles en fonction de l'élément sélectionné dans l'arborescence.
  • Affichage
    • Barre d'outils
      Affiche ou non la barre d'outils sous la barre de menu.
    • Section modulateurs
      Affiche ou non la section permettant l'édition des modulateurs.
    • Clavier
      Modifie la longueur du clavier virtuel.
  • Aide
    • Manuel
      Affiche ce manuel.
    • A propos
      Affiche la version du logiciel.
