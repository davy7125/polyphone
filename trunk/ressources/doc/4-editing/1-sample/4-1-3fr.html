L'égaliseur permet d'augmenter ou diminuer certaines fréquences, de -15 à +15 dB. L'effet s'applique après appui sur le bouton « appliquer ». Un appui sur « réinitialiser » remet tous les boutons à 0 dB (mais n'enlève pas l'effet sur le son : pour cela utiliser le bouton « annuler » dans le menu « Édition »).

Chaque modification d'un son stéréo peut être répercutée sur le son lié si l'option est cochée dans les préférences.
